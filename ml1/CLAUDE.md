# ML1: Introduction to Machine Learning

This directory contains the first machine learning course focusing on foundational concepts.

## Structure

Contains 11 lectures covering introductory machine learning topics:
- lecture01, lecture02, lecture03, lecture04, lecture05
- lecture06, lecture07, lecture10, lecture12, lecture13, lecture14

## Content Format

Each lecture directory contains:
- `index.html` - Main lecture content and presentation

## Course Focus

ML1 covers fundamental machine learning concepts including:
- Basic mathematical foundations
- Core ML algorithms and principles
- Simple model implementations
- Introduction to model evaluation
