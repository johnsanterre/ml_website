# ML Topics Library

This directory contains standalone machine learning topic modules.

## Structure

Contains 10 focused topic modules:
- topic01 through topic10
- python/ - Python-related ML content

## Content Format

Each topic directory contains:
- `index.html` - Topic presentation and content

## Purpose

These are modular, self-contained topics that can be used:
- As supplementary material for ML1 and ML2 courses
- As standalone reference material
- For targeted learning on specific ML concepts
- As building blocks for custom course arrangements

## Related Content

- More extensive topic coverage available in `/topics/` directory
- Generated PDFs stored in `/output/` when applicable
