2. Slide Templates

a) Title Slide Template:
```html
<section class="title-slide">
    <h1>Main Title</h1>
    <p>Subtitle or Additional Information</p>
</section>
```

b) Single Focus Slide Template:
```html
<section>
    <h2>Topic Title</h2>
    <div class="image-container">
        <img src="path/to/image.jpg" alt="Description">
    </div>
    <p>Key message or explanation</p>
</section>
```

c) Two-Column Slide Template:
```html
<section>
    <h2>Two Column Layout</h2>
    <div class="two-column">
        <div class="column">
            <ul>
                <li class="fragment">Point 1</li>
                <li class="fragment">Point 2</li>
                <li class="fragment">Point 3</li>
            </ul>
        </div>
        <div class="column">
            <div class="image-container">
                <img src="path/to/image.jpg" alt="Description">
            </div>
        </div>
    </div>
</section>
```

d) Content with Footnote Template:
```html
<section>
    <h2>Content with Reference</h2>
    <p>Main content with reference<span class="footnote-ref">1</span></p>
    <div class="footnote">
        <p>1. Reference details or additional information</p>
    </div>
</section>
```

3. Usage Guidelines

a) Image Best Practices:
- Use high-resolution images (minimum 1920x1080)
- Optimize images for web delivery
- Maintain consistent aspect ratios
- Use transparent backgrounds when possible
- Include meaningful alt text

b) Content Structure:
- Keep one main point per slide
- Use consistent heading hierarchy
- Limit bullet points (3-5 per list)
- Use fragments for complex explanations
- Include visual aids when possible

c) Typography:
- Main titles: 2.5em
- Subtitles: 1.8em
- Body text: 1em
- Footnotes: 0.25em
- Use consistent text alignment

d) Presentation Flow:
- Start with agenda/overview
- Group related topics
- Include transition slides
- End with summary/key takeaways
- Add Q&A slide at the end

4. Accessibility Considerations:
- Use sufficient color contrast
- Include alt text for images
- Maintain readable font sizes
- Ensure keyboard navigation
- Test with screen readers

5. Technical Setup:
- Test all transitions
- Verify fragment animations
- Check responsive layout
- Ensure proper PDF export
- Test offline functionality
